POS Order Line Employee Assignment

Assign Staff, Track Performance, and Confirm Sales on Receipts

This module enhances the Point of Sale (POS) interface by adding a **dedicated button** to select and assign the employee responsible for **each individual order line**.

This functionality ensures detailed sales tracking, facilitates accurate commission calculation, and provides full accountability for every line item processed.

Key Features

  • **Assign Employee per Line:** A custom button/control is added to the POS screen to link a staff member to the sale item.
  • **Receipt Confirmation:** The selected employee's name is **printed on the POS receipt** next to the line item, providing documentation of who processed the sale.
  • **Backend Tracking:** Employee ID is saved on the pos.order.line model, ensuring the operation is documented in the Odoo backend for reporting.
  • **Minimal Dependencies:** Only requires the **Point of Sale** and **HR (Human Resources)** modules to function.
